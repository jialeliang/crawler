# Crawler

方便快捷的爬虫管理平台，通过简单的配置实现定制数据。

技术栈使用：Egg、Vue

前端模板使用：Vue-admin-template



## 启动项目

下载后，命令行输入

```shell
// 安装依赖
> npm install
// 启动开发服务
> npm run dev

// 进入前端
> cd vue-admin-template
> npm install
> npm run dev
```
